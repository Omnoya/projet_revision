WCS_partage
===========

####Description
Projet de révision

#### Pré-requis
Avoir "composer" d'installé => https://getcomposer.org

1. **SSH** git@github.com:Omnoya/projet_revision.git
2. **HTTPS** https://github.com/Omnoya/projet_revision.git
3. cd projet_revision
4. composer install
5. php app/console doctrine:database:create
6. php app/console doctrine:schema:update --force
7. php app/console asset:install
